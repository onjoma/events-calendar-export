Export your The Events Calendar events to beautifully formatted PDF documents.

== Description ==

LindaWP Events Export is a powerful WordPress plugin that allows you to export your events from The Events Calendar to PDF format. Perfect for creating event brochures, sharing offline event information, or archiving event details.

= Features =

* Export events to PDF with a single click
* Select date range for export
* Includes event details such as:
  * Event title and description
  * Date and time information
  * Location details
  * Featured images
* Clean, professional PDF layout
* Supports internationalization
* Compatible with The Events Calendar plugin

= Requirements =

* WordPress 5.2 or higher
* PHP 7.2 or higher
* The Events Calendar plugin installed and activated

== Installation ==

1. Upload the `lindawp-events-export` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Events Export' in your admin menu to use the plugin

== Frequently Asked Questions ==

= Does this plugin work without The Events Calendar? =

No, this plugin requires The Events Calendar plugin to be installed and activated.

= Can I customize the PDF layout? =

Currently, the PDF layout is fixed, but we plan to add customization options in future updates.

= What information is included in the PDF? =

The PDF includes event title, description, featured image, date, time, and location information.
